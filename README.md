idea-marginizer
===============

IDEA Plugin for align key = value pair

```scala
// before
val test = "Hello world!"
val mySuperValue = 123
val testActor = actorSystem.actorFor(Props(new Producer))

// after
val test         = "Hello world!"
val mySuperValue = 123
val testActor    = actorSystem.actorFor(Props(new Producer))
```
